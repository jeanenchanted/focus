# .setcolour

### Usage

> .setcolour \[colour.name\]

`[colour.name]` The name of the colour.

{% hint style="info" %}
[Here](https://www.w3schools.com/colors/colors_names.asp) is a list of valid colour names.
{% endhint %}

{% hint style="warning" %}
#### Don't include these symbols when running the command.

`{} -`optional argument  
`[] -` required argument
{% endhint %}
