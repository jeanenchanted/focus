# .setranking

### Usage

> .setranking \[message.string\]

### Aliases

>

`[message.string]` The level-up message.

{% hint style="info" %}
#### Available Functions

`{user} -` your username  
`{mention} -` your mention  
`{level} -` your new level
{% endhint %}

{% hint style="warning" %}
#### Don't include these symbols when running the command.

`{} -`optional argument  
`[] -` required argument
{% endhint %}
