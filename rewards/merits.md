# .merits

### Usage

> .merits {user.mention}

`{user.mention}` The credits of whom you are checking. If none, defaults to your current user.

{% hint style="warning" %}
#### Don't include these symbols when running the command.

`{} -`optional argument  
`[] -` required argument
{% endhint %}
