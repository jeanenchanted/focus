# .gift

### Usage

> .gift \[user.mention\] \[amount.number\]

`[user.mention]` The person you are giving credits to.  
`[amount.number]` How many credits you are giving.

{% hint style="warning" %}
#### Don't include these symbols when running the command.

`{} -`optional argument  
`[] -` required argument
{% endhint %}
