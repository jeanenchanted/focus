# .rank

### Usage

> .rank {user.mention}

`{user.mention}` The rank of whom you are checking. If none, defaults to your current user.

{% hint style="warning" %}
#### Don't include these symbols when running the command.

`{} -`optional argument  
`[] -` required argument
{% endhint %}
